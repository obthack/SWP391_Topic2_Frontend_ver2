🔧 Fix Login Issue

Sửa lỗi không đăng nhập được

⚠️ Vấn đề

Token đã hết hạn và hệ thống cố gắng refresh nhưng backend chưa hỗ trợ endpoint refresh. Điều này gây ra loop và không cho phép đăng nhập lại.

✅ Giải pháp

Đã fix code:

  1. Vô hiệu hóa auto token refresh (backend chưa có endpoint)
  2. Không tự động xóa token khi expired (để backend xử lý 401)
  3. Cho phép login lại bằng cách skip token check cho login endpoint

🔨 Hành động ngay:

Click nút bên dưới để xóa token cũ và đăng nhập lại:

📝 Các thay đổi đã thực hiện:

// tokenManager.js - Disabled auto token refresh (backend không hỗ trợ) - Token expired sẽ vẫn được trả về, để backend response 401 - Không tự động clear auth khi token expired // api.js - Skip token refresh cho login/register endpoints - Chỉ clear auth khi thực sự nhận 401 từ backend - Không redirect nếu đang ở trang login

🎯 Sau khi fix:

  1. Click "Xóa Token Cũ" ở trên
  2. Reload trang web (F5)
  3. Đăng nhập lại bình thường
  4. Token mới sẽ hoạt động cho đến khi backend response 401

💡 Lưu ý:

Token Lifetime: Token hiện tại sẽ hoạt động cho đến khi backend thực sự từ chối (401). Khi đó user cần đăng nhập lại.

Future Enhancement: Khi backend có endpoint /api/auth/refresh, uncomment code trong tokenManager.js để enable auto refresh.

🔧 Format cuối cùng (ĐÚNG)

✅ JSON Format đúng cho POST:

{ "request": { "orderId": 23, "revieweeId": 1, "rating": 5, "content": "được" } }

✅ Code trong MyPurchases.jsx:

const requestData = { orderId: selectedProduct.orderId, revieweeId: selectedProduct.sellerId || selectedProduct.seller?.id || 1, rating: reviewData.rating, content: reviewData.comment }; await apiRequest('/api/Review', { method: 'POST', body: JSON.stringify({ request: requestData }) });

✅ So sánh với PUT endpoint:

// POST /api/Review - CẦN wrapper { "request": { "orderId": 23, "revieweeId": 1, "rating": 5, "content": "được" } } // PUT /api/Review/{id} - KHÔNG CẦN wrapper { "orderId": 23, "revieweeId": 1, "rating": 5, "content": "được" }

🧪 Test Results

✅ API Test Results:

  • Format không có wrapper: ❌ 400 Bad Request
  • Format có wrapper "request": ✅ 401 Unauthorized (format đúng, cần token)

🎯 Kết quả:

Format với wrapper `{"request": {...}}` là đúng cho POST endpoint. Lỗi 401 là bình thường vì cần JWT token.
